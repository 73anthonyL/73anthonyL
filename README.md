# Anthony Lu

EECS student @ UC Berkeley building applied AI research projects, robotics software, and developer tools.

## Selected work

### Social Media Extremism Detection

Reproducible NLP research repository for binary classification of social-media text as extremist or non-extremist. The project includes a curated dataset workflow, fixed train/validation/test splits, baseline model comparisons, standardized result summaries, and responsible-use documentation.

**Areas:** NLP, machine learning, text classification, interpretability, research engineering  
**Repository:** https://github.com/73anthonyL/extremism_sentiment_analysis

### FRC robot software

Robot software developed with Firebots Software for FIRST Robotics Competition seasons. These repositories show collaborative Java development, WPILib-based robot architecture, and iterative engineering under competition constraints.

**Areas:** robotics, Java, WPILib, controls, team software engineering  
**Repositories:**

- https://github.com/firebots-software/2026-Rebuilt
- https://github.com/firebots-software/2025-reefscape
- https://github.com/firebots-software/2024-crescendo

### Competitive Programming

C++ competitive-programming archive with reusable snippets, practice solutions, and algorithm notes.

**Areas:** C++, data structures, graph algorithms, number theory, contest programming

### Reconnaissance

**Areas:** Python, CLI tooling, information retrieval, structured scoring, product prototyping  
**Repository:** https://github.com/73anthonyL/reconnaissance
